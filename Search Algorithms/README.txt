Sliding Block Puzzle problem, using various search algorithms:   
1.) Random Walk Generation
2.) Breadth First Search
3.) Depth First Search
4.) DFS with Iterative Deepening
5.) A* Search    

How to Run:    
-------------    
1.) On the command line, issue "make" and the executable "sbp" will be produced along with output of code in "output-sbp.txt".    
2.) Simply type ./sbp onto the command line to run the program    
3.) Source code is all stored in main.cpp
